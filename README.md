# AiSch_core_project
인공지능사관학교 웹 js특화과정 따숨 팀 - 니어케어 프로젝트

 
### 깃허브 & 로컬 연결 방법
로컬에서 원하는 폴더 지정하고 vscode 열기 -> 터미널에에 git clone https://github.com/memorygreen/AiSch_core_project.git 입력

### npm 패키지 설치
npm i
